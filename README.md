AngularJS CRUD App
================

A sample CRUD application by AngularJS with Ruby on Rails

## Installation

```
$ git clone git@github.com:jwako/angular_crud_app.git
```

## How to start

```
$ cd angular_crud_app
$ rake db:create
$ rake db:migrate
$ rails s
```

## Environments

- Ruby 2.0
- Rails 4.1.0
- AngularJS 1.2.21
